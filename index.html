Static Assets

This server hosts static assets for OEASE applications and services. All resources are served with appropriate caching headers and optimized for fast delivery.

URL Structure

https://assets.oease.net/[asset-type]/[version]/[filename]

Available Asset Types

JS

JavaScript

Client-side scripts and application code including libraries and frameworks.

CSS

Stylesheets

CSS files including base styles, themes, and component-specific styles.

IMG

Images

Optimized images in various formats: PNG, JPG, SVG, and WebP.

DOC

Documents

PDF documentation, manuals, and other static document resources.

FNT

Fonts

Web fonts in WOFF and WOFF2 formats with appropriate cross-origin headers.

MOD

Modules

ES modules and other modular components with versioning support.
